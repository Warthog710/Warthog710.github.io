Background

Avinash Agrawal, senior director engineering for the industrial edge computing business of Dialog Semiconductors manages a team that has worked on an edge server called SmartServerIoT that is designed to be used in environments of hundreds to thousands of devices. This HW must be tested to maximize the number of supported devices at a time and prove stability.

Scope

The project proposed will interact with the SmartServer by simulating devices and communicating over twisted pair using the LONTALK protocol. The simulator produced will assist in testing the stability of the SmartServer while under high-Load. The ultimate goal of this project is to be able to simulate 2k devices on a single simulator.

Proposed Solution

Produce a software simulator that runs on a Linux machine that is capable of simulating devices. This SW will interact with the SmartServer sending device parameters over a pre-defined protocol.

Highlights

In this project, we will be producing a piece of software that interacts with real world hardware. While Dialog already has a simulator capable of performing this task, their existing solution is lacking several key features. Our proposed solution will be capable of simulating multiple unique device types at the same time, run on a Linux machine and scale to a larger pool of devices.
